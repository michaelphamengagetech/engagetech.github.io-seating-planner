How to get the correct URL
  1. Open your Google Sheet
  2. Click File → Share → Publish to web
  3. In the first dropdown select your People tab (not "Entire Document")
  4. In the second dropdown select Comma-separated values (.csv)
  5. Click Publish → copy the URL
  6. The URL must end in pub?output=csv or &output=csv
Common mistakes
✗ URL ends in pubhtml — wrong, change to pub?output=csv
✗ Using the regular share link — must be File → Publish to web
✗ Sheet is not public — set to "Anyone with the link can view"
Test your URL first
Paste your URL in a new browser tab — you should see plain comma-separated text, not a formatted sheet.
